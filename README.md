# Viva Fit — Projeto pronto para rodar (Firebase placeholders)

Este projeto foi preparado para inicializar o Firebase (Auth, Database, Firestore, Storage).
Edite `src/environments/environment.ts` e cole as credenciais do seu projeto Firebase.

## Como usar

1. Instale dependências (recomendado Node LTS 18/20):
```bash
cd viva-fit-main
npm install
```

2. Rode o app com Ionic:
```bash
npm install -g @ionic/cli
ionic serve
# ou
npm start
```

3. Se houver erros de peer dependency, tente:
```bash
npm install --force
```

## Observações
- `environment.ts` contém placeholders. Substitua pelos valores do seu Firebase Console.
- Este projeto usa as versões compatíveis com Angular 20 e @angular/fire compat layer.

- Para baixar o Apk siga essa linha aqui
-viva-fit-main\android\app\build\outputs\apk\debug
